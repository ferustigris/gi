Генератор интерфейсов

Программа предназначена для генерации интерфейсов на ПЭВМ по запросу с ПЛИС по протоколам стека TCP/IP.

При запуске программы в системном трее появляется значек в виде кисточки. При нажатии по нему правой клавиши мыши, появляется контекстное меню, содержащее пункты

  1. Показать журнал;

  2. Очистка;

  3. Помощь;

  4. Выход.

При выборе пункта Показать журнал отображается окно с журналируемой информацией.

При выборе пункта Очистка удаляются все виджеты, созданные ранее.

При выборе пункта Помощь отображается данная информация.

При выборе пункта Выход осуществляется выход из программы.

Протокол

Управляющее слово передается в составе пакета UDP (стека протоколов TCP/IP) в поле данных, начиная с 1 по конечный байт. Соединение устанавливается на порт 45454. Идентификация виджета осуществляется исходя из последнего байта IP-адреса отправителя (ПЛИС), порта отправителя и ИД - Идентификатора виджета. Управляющее слово состоит из:

1.      Идентификатора виджета;

2.      Типа виджета;

3.      Номера команды;

4.      Данных;

Таким образом, структура команды имеет вид

Идентификатор виджета

Тип виджета

Команда

Данные

1 байт

1 байт

1 байт

1…N байт

Где

 

- Идентификатор виджета (1 байт)

8 битовое число, определяющее индивидуальный номер виджета;

 

- Тип виджета (1 байт)

0х00 – текстовое поле, принимающее текст и 32 битные числа;

0х01 – текстовое поле, принимающее числа с плавающей точкой длиной 32 бита;

0х02 – текстовое поле, принимающее целые 32 битные числа;

0х03 – график;

0х04 – текстовое поле, принимающее текст и 32 битные числа;

0х05 – поле ввода 32 битного числа с плавающей запятой;

0x06 — цветовой индикатор;

0x07 — аналоговый индикатор типа спидометр;

0x08 — панель, объединяющая другие виджеты и панели;

0x09 — текстовое поле, принимающее числа с плавающей точкой длиной 32 бита, отображающееся как ЖК табло;

0x0A — поле ввода 32 битного целого числа;

 

- Команда (1 байт)

0х00 – установить координаты и размер;

0х01 – установить значение;

0х02 – закрыть (скрыть) виджет;

0х03 – показать виджет;

0х04 — установить заголовок виджета;

 

- Данные (n байт)

структура данных зависит от команды и типа виджета.
